# 🧠 Deep Learning Lab Practicals

This repository contains the implementation of **Deep Learning Laboratory Experiments** using Python and TensorFlow/Keras. The practicals cover fundamental and advanced deep learning concepts, including Deep Feed Forward Networks (DFFN), Convolutional Neural Networks (CNN), Recurrent Neural Networks (RNN/LSTM), Transfer Learning, Autoencoders, and Generative Adversarial Networks (GANs).

---

## 📚 List of Experiments

### 1. Introduction to Deep Learning Frameworks
- TensorFlow/Keras Installation
- Basic Neural Network Creation
- Model Compilation and Training

### 2. Housing Price Prediction using Deep Feed Forward Network (DFFN)
- Data Preprocessing and Normalization
- Deep Neural Network for Regression
- Performance Evaluation using MAE, MSE, and RMSE

### 3. Classification using Deep Feed Forward Network
- Data Preprocessing
- Deep Neural Network Classifier
- Accuracy and Confusion Matrix Analysis

### 4. Regularization Techniques for Deep Learning
- L1 and L2 Regularization
- Dropout Implementation
- Overfitting Analysis

### 5. Hyperparameter Tuning and Model Optimization
- Learning Rate Tuning
- Batch Size Optimization
- Optimizer Comparison
- Activation Function Analysis

### 6. Handwritten Digit Recognition using CNN
- MNIST Dataset
- Convolutional Neural Network (CNN)
- Model Training and Prediction

### 7. Dog Breed Classification using Transfer Learning
- Transfer Learning using VGG16, ResNet50, and MobileNet
- Fine-Tuning Pretrained Models
- Classification Performance Evaluation

### 8. Stock Market Prediction using Recurrent Neural Networks (RNN/LSTM)
- Sequential Data Preparation
- LSTM Model Implementation
- Stock Price Prediction and Visualization

### 9. Slot Filling using Recurrent Neural Networks
- Text Preprocessing
- Sequence Labeling using RNN/LSTM
- Prediction Accuracy Evaluation

### 10. Credit Card Fraud Detection using Deep Learning
- Handling Imbalanced Datasets
- Deep Neural Network Classifier
- Precision, Recall, and F1-Score Evaluation

### 11. Convolutional Autoencoder on MNIST Dataset
- Encoder–Decoder Architecture
- Image Reconstruction
- Original vs Reconstructed Image Comparison

### 12. Generative Adversarial Network (GAN)
- Generator and Discriminator Implementation
- GAN Training
- Synthetic Handwritten Digit Image Generation
